# New Training Slides Outline: Day 1 + Day 2 (Pre-Fork)
## For PPTX Build + Learning Hub Module

**Purpose:** This outline replaces the current All-in-One deck (56 slides) and the first portion of the Advanced System deck. It is designed to be:
1. Built as a PPTX slide deck for in-person training
2. Fed to Claude Code to build as a Learning Hub self-paced module
3. Interactive, visual-first, and minimal passive reading

**Design Principles:**
- No slide should be "wall of text" — every slide has ONE clear point
- Every module ends with an interaction (mini-quiz, activity, or discussion prompt)
- KB article links embedded throughout (builds the habit)
- Video markers 🎬 indicate where a short video (1-2 min) would be ideal
- Trainer-led markers 👩‍🏫 indicate where trainer drives (not self-study)
- Self-study markers 📖 indicate where player reads/explores independently
- Interactive markers 🎮 indicate quiz, activity, or hands-on element

**Estimated total slides:** ~65 across both days (vs. 56 in old deck that covered only Day 1 content)
